# Sistem lozinke – Flowcode

Ovaj projekat predstavlja simulaciju sistema lozinke urađenog u programu Flowcode. Cilj projekta je prikaz rada jednostavnog sigurnosnog sistema pomoću unosa i provjere šifre.

Sistem funkcioniše kroz više stanja:

- Unos lozinke
- Provjera tačnosti unosa
- Dozvola pristupa pri tačnoj šifri
- Odbijanje pristupa pri pogrešnoj šifri

Projekat je napravljen kao vježba za:

- digitalnu logiku
- automatiku
- simulaciju mikrokontrolera
- rad sa uslovima i kontrolom pristupa

## Tehnologije

- Flowcode
- Mikrokontroler (zavisno od konfiguracije projekta)
- Tastatura ili tasteri za unos lozinke
- LED diode
- LCD displej

## Funkcionalnosti

- Unos korisničke šifre
- Provjera ispravnosti lozinke
- Signalizacija tačnog i netačnog unosa
- Jednostavan sigurnosni sistem

## Pokretanje projekta

1. Otvoriti projekat u programu Flowcode.
2. Pokrenuti simulaciju.
3. Unijeti lozinku pomoću tastera ili tastature.
4. Posmatrati reakciju sistema.
